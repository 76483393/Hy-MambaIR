# Demo Data

Only representative demo patches should be placed here.

## Current demo subset

This directory currently contains four curated LR/HR demo patch pairs with shared basenames across the two folders.

These demo patches were refreshed from the validated REV patch subset using the current released evaluation pipeline to provide a stronger public demonstration while keeping the package compact.

- `demo_001.tiff` ← `REV1760_patch_02`
- `demo_002.tiff` ← `REV1368_patch_00`
- `demo_003.tiff` ← `REV1460_patch_02`
- `demo_004.tiff` ← `REV1371_patch_00`

## Do not place here

- raw 3D CT volumes
- full REV data
- sample metadata with restricted acquisition details
