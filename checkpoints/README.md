# Checkpoints

This directory stores released public checkpoints only.

## Released checkpoint

The current public repository provides one released main checkpoint:

- `Hy-MambaIR_x4_main.pth`

This checkpoint supports evaluation and smoke testing of the released x4 setup under the provided pipeline.

## Scope

The public checkpoint package is intentionally minimal. It does not include:

- raw training states
- unrelated historical checkpoints
- baseline model checkpoints from other methods

If additional artifacts are needed for technical verification, please contact the project maintainers.
